# Cucumber with JS
Special credits for [`nightwatch-cucumber`](https://github.com/mucsi96/nightwatch-cucumber)
## How to Use



```
npm i -g ntl
git clone git@github.com:fathirwafda/pirate-ship.git
cd pirate-ship
npm i

```
Execute the test
```
ntl
or run using tag
npm run e2e-test -- --tag test
```
